# 🎬 Netflix Data Analysis (EDA Project)

## Overview
This project explores the Netflix dataset to understand content distribution, growth trends, and popular genres using Exploratory Data Analysis (EDA).

## Objectives
- Analyze distribution of Movies vs TV Shows
- Study content growth over the years
- Identify top genres on Netflix
- Explore runtime distribution of movies
- Generate top-10 insights (genres, years)

## Analysis Performed

### 1. Content Type Distribution
- Compared number of Movies vs TV Shows
- Movies dominate the platform, indicating a stronger focus on film content

### 2. Year-wise Growth
- Analyzed number of releases over time
- Observed rapid growth in recent years, showing expansion of Netflix's library

### 3. Genre Analysis
- Extracted and analyzed genres from the dataset
- Drama and Comedy are the most common genres, reflecting audience preferences

### 4. Runtime Distribution
- Focused on movie durations
- Most movies fall within a standard range (80–120 minutes)

### 5. Top Insights
- Identified top 10 genres
- Identified top 10 release years
